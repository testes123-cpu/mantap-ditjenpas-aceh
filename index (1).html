Panduan Setup Google Form + Spreadsheet
Langkah-langkah mengintegrasikan MANTAP ke ekosistem Google
✅ Setelah setup selesai, semua data dari Google Form akan otomatis masuk ke Google Spreadsheet secara real-time.
Langkah setup
1
Buat Google Spreadsheet sebagai database pusat
Buka sheets.google.com → buat spreadsheet baru → beri nama MANTAP Kanwil Ditjenpas Aceh 2026.
Buat 5 tab: Dashboard, Cek Harian, Laporan Gangguan, Inventaris, Berita Acara.
2
Buat 4 Google Form sesuai komponen MANTAP
Buka forms.google.com → buat 4 form: Form Cek Harian, Kartu Lapor Gangguan, Buku Inventaris, dan Berita Acara.
Aktifkan Collect email addresses di Settings agar pelapor teridentifikasi.
3
Hubungkan setiap Form ke tab Spreadsheet
Di setiap Google Form → tab Responses → klik ikon spreadsheet hijau → pilih Select existing spreadsheet → pilih file MANTAP → pilih tab yang sesuai. Semua respons otomatis masuk.
4
Aktifkan notifikasi email otomatis
Di Spreadsheet → menu Tools → Notification rules → pilih Any changes are made → masukkan email pengelola fasilitas → simpan. Pengelola akan ternotifikasi setiap ada laporan masuk.
5
Buat dashboard otomatis dengan formula
Di tab Dashboard, gunakan formula berikut untuk ringkasan otomatis:
=COUNTIF('Cek Harian'!D:D,"Baik") ← Hitung kondisi Baik =COUNTIF('Laporan Gangguan'!F:F,"Tinggi") ← Laporan urgensi tinggi =COUNTA('Inventaris'!B:B)-1 ← Total aset tercatat =COUNTIF('Berita Acara'!J:J,"Selesai sempurna") ← Perbaikan selesai
6
Pasang Apps Script untuk notifikasi WhatsApp (opsional)
Di Spreadsheet → menu Extensions → Apps Script → tempel script berikut:
function onFormSubmit(e) { var sheet = e.source.getActiveSheet(); var row = e.values; var subject = "🔔 Laporan Gangguan Baru - MANTAP"; var body = "Fasilitas: " + row[3] + "\n" + "Masalah: " + row[4] + "\n" + "Urgensi: " + row[5] + "\n" + "Pelapor: " + row[1] + "\n" + "Tanggal: " + row[0]; MailApp.sendEmail("email_pengelola@gmail.com", subject, body); }
Link penting
→ Google Forms (buat form baru) → Google Sheets (buat spreadsheet) → Google Drive (kelola semua file)
